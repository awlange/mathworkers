(static) summary(data) → {StatisticsSummary}

Compute basic summary statistics for a generic Array, Float64Array, Vector, or Matrix Returns an object containing number of elements, mean, standard deviation, minimum, maximum, and quartiles. The quartiles computed here are the so-called "Tukey's Hinges". Quartiles are not reported if the data passed in contains less than 3 elements
